# Jinja2 模板结构说明

使用以下数据结构来填充 `context_template` 的模板内容：

```json
{
    "application": {
        "name": "text",
        "description": "text",
        "terms": [
            {
                "term": "text",
                "defination": "text"
            }
        ],
        "pages": [
            {
                "name": "text",
                "description": "text",
                "features": [
                    {
                        "name": "text",
                        "description": "text",
                        "document": "text"
                    }
                ]
            }
        ]
    }
}
```
